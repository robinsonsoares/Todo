 Criar os projetos:
 - Todo.Domain: nunca referencia, ele é sempre referenciado
 - Todo.Domain.Api: referencia o Domain e Domain.Infra 
 - Todo.Domain.Infra: referencia o Domain
 - Todo.Domain.Tests: referencia o Domain
 Comando para referenciar: "dotnet add reference <Pasta do Projeto>"
 
 Criar uma SOLUTION: dotnet new sln 
 Adicionar os projetos na SOLUTION: dotnet sln add  <Projetos>
